**NYC Dashboard — Power BI Project**
 
**Project Overview**
This project is an interactive Power BI dashboard that provides deep insights into key indicators related to New York City data.
The dashboard is designed to help decision-makers understand patterns, trends, and performance metrics through clean visual storytelling.


**Project Objectives**
Analyze important city metrics and trends
Discover patterns and anomalies in the data
Present insights in a simple, interactive, and visually appealing way
Support data-driven decision making

**Dataset Description**
The dataset contains structured information related to New York City activities and performance.
It includes multiple variables representing different dimensions of the city’s operations and services.
(Exact column details are available inside the Power BI model)

**Tools & Technologies**
Power BI Desktop
Power Query for data cleaning & transformation
DAX for calculated measures
Data Visualization & Dashboard Design


**Key Features**
Interactive filters & slicers
KPI cards showing main performance indicators
Trend analysis using charts and graphs
Clean and user-friendly dashboard layout
Drill-down & cross-filtering capabilities


**Key Insights & Analysis**
The dashboard enables:
Identification of high and low performance areas
Trend analysis across time periods
Comparative analysis between different categories
Quick monitoring of essential KPIs


**Project Workflow**
Data Collection & Import
Data Cleaning & Transformation using Power Query
Data Modeling
Measure creation using DAX
Dashboard design & visualization
Insight generation & validation


**How to Use**
Download the file: NYC - Dashboard.pbix
Open with Power BI Desktop
Refresh data if needed
Interact with the filters and visuals to explore insights
